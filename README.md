# Data Jobs Dashboard w/ Power BI
![alt text](images/Project1_Dashboard_Overview.gif)
## Introduction
Navigating the 2024 data job market shouldn't feel like guesswork. This dashboard was built to consolidate scattered market data into a clear, unified view. Whether you are **transitioning careers or negotiating a new salary, this tool allows you to explore real-world compensation and location trends across the data science landscape** in one easy-to-use interface.
### Dashboard File
You can find the file for the dashboard here: [Data_Jobs_Dashboard.pbix](Data_Jobs_Dashboard.pbix)
## Skills Showcased
This project served as a comprehensive deep-dive into Power BI's end-to-end analytics capabilities. Key technical milestones included:

- **End-to-End ETL**: Performed data orchestration in Power Query, including schema standardization, null handling, and custom column logic.

- **Advanced DAX & Analytics**: Developed measures to calculate core business KPIs such as Median Yearly Salary and Total Job Volume.

- **Geospatial & Trend Mapping**: Visualized global market distribution via Map Charts and analyzed temporal shifts using Line and Area Charts.

- **UX/UI Optimization**: Applied dashboard design principles to create a cohesive "data story," utilizing Bookmarks, Buttons, and Slicers for a web-app-like user experience.

- **Granular Drill-Throughs**: Engineered drill-through pathways to allow users to move seamlessly from high-level trends to row-level details.
## Dashboard Overview
This report is split into two distinct pages to provide both a high-level summary and a detailed analysis.
### Page 1: High-Level Market View

Your Mission Control for the data job market. This dashboard eliminates the noise by highlighting the metrics that matter most: job availability, salary benchmarks, and the most in-demand roles. Get a comprehensive market pulse in seconds.
### Page 2: Job Title Drill Through
![alt text](images/Project1_Dashboard_Page2.gif)
Deep-Dive Capabilities Navigate from the big picture to the fine details with a single click. The Drill-Through page acts as a deep-dive tool, allowing you to isolate a specific job title to view its unique salary landscape, remote-work prevalence, and global hiring hotspots.
## Conclusion
This dashboard showcases how Power BI can transform raw job posting data into a powerful tool for career analysis. It allows users to slice, filter, and drill through data to make informed decisions about their career paths.
